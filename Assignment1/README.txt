there are 4 commmand line variables - order, size, algorithm, outputfile

order:
      - "increasing"
      - "decreasing"
      - "random"
     
size:
      - size of the array wanted
      - 10,100,10000,etc.
 
algorithm:
      - "selection"
      - "quick"
      - "insertion"
      - "merge"
  
outputfile:
      - name of the output file wanted (.txt)
      

Example Command line input:
      - javac Assign1.java
      - java Assign1 increasing quick 1000 test.txt
